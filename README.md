# Conway's Game of Life

## Introduction 

net7.0 Console application that simulates [Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life)

# Getting Started

 ### Software dependencies

 - [.Net 7.0](https://dotnet.microsoft.com/en-us/download/dotnet/7.0)

### From cmd line
- `git clone https://github.com/PavoGrande/GameOfLife.git`
- `cd .\GameOfLife`
- `dotnet build .\src`

## Run the App

### From cmd line:

- `dotnet run --project .\src\GameOfLife\GameOfLife.csproj`
